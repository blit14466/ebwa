Privacy Policy

We recommend users enter as little personal information as possible: only what's necessary to find penpals. We do not collect data beyond that required to run the site. Nor do we sell or trade user data.

We process and store the following types of (potentially personal) data:

Type Description Used for Shared off-site
Account data username, password basic site services no
Cookies stored on your computer keeping track of your login status no
Social login token only if you use social login (like "sign in with google") login with your login provider
Email address optional basic site services, notifications (if enabled by user), password reset with our mail provider (if notifications enabled)
Ad data The data you enter under "Edit ad" basic site services no
Message data Messages you send and receive on our site basic site services no
Technical data IP address, timestamps, referer, error reports, debug logs, ...
if necessary other security-relevant data		 can be used for fixing bugs, stopping abuse and ensuring network and information security no

We also use aggregated, anonymized forms of some of the data above to statistically analyze site usage for development purposes - e.g. to see what features get used or which browsers we should test changes with.

You can:
  • edit/delete data for your ad
  • export your data
  • delete your account
  • remove or limit your consent to this policy (equivalent to removing your account)

Removed data might remain on our servers for up to a month until it is permanently deleted. (Data might be kept longer if required for security or legal reasons.)

Data might be shared beyond what is outlined here if required by law.

We will provide prominent notice of any (non-cosmetic) changes to this privacy policy.
